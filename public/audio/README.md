# Background Music Setup

## Instructions

1. Place your `.wav` music file in this directory
2. Name it `background.wav` (or update the path in `BackgroundMusic.jsx`)
3. The music will auto-play when users interact with the site

## Supported Formats
- WAV (recommended for quality)
- MP3 (for smaller file size)
- OGG (for web compatibility)

## File Location
`m:\NoveltyCams\public\audio\background.wav`
